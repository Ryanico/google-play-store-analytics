# google-play-store-analytics
Exploratory data analysis of Google Play Store apps and user reviews using Python, Pandas, and Plotly to uncover trends, ratings patterns, and market insights.

# 📊 Google Play Store App Analytics

This project performs an in-depth exploratory data analysis (EDA) of the Google Play Store dataset to uncover trends, patterns, and insights about Android applications.

## 🚀 Project Overview

The goal of this project is to analyze thousands of apps on the Google Play Store and answer key questions such as:

- What categories dominate the Play Store?
- How do ratings vary across app types?
- What factors influence app popularity?
- How do free vs paid apps compare?

## 📂 Dataset

- **Source:** [Kaggle - Google Play Store Apps Dataset](https://www.kaggle.com/lava18/google-play-store-apps)
- Includes:
  - App details (name, category, rating, installs, price, etc.)
  - User reviews

## 🛠️ Technologies Used

- Python 🐍
- Pandas
- Plotly (for interactive visualizations)
- Jupyter Notebook

## 📈 Key Analysis Performed

- Data cleaning and preprocessing
- Category-wise app distribution
- Rating and review analysis
- Free vs Paid app comparison
- Install trends and popularity insights
- Visualization of key metrics

## 📊 Sample Insights

- Certain categories dominate in terms of app count
- Free apps significantly outnumber paid apps
- Higher installs don’t always guarantee higher ratings
- Pricing strategies vary across categories
